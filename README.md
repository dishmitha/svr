# 📈 Support Vector Regression (SVR) Project

A Machine Learning project implementing Support Vector Regression (SVR) for predicting continuous numerical values using regression techniques.

---

# 📌 Project Overview

This project demonstrates the implementation of Support Vector Regression (SVR) including data preprocessing, feature scaling, model training, prediction, evaluation, and deployment.

The project includes:

- Data preprocessing
- Handling missing values
- Feature scaling
- Model training
- Prediction system
- Model evaluation
- Data visualization
- Streamlit deployment

---

# 🤖 Algorithm Used

## Support Vector Regression (SVR)

SVR is a supervised Machine Learning regression algorithm used for predicting continuous numerical values.

Unlike traditional regression models, SVR tries to fit the best boundary within a threshold margin.
